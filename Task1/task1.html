Answers

  1. What's the difference between a normal div and a grid container?
    A normal div stacks children vertically. A grid container can arrange children in rows and columns.
  2. Why doesn't the layout change immediately?
    Grid needs grid-template-columns or grid-template-rows to define the structure.
  3. What is required for Grid to show visible changes?
    You need to define columns/rows using properties like grid-template-columns.

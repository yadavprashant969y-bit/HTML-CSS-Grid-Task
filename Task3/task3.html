Answers

  1. Which layout breaks faster on small screens?
    Fixed layout (px) breaks faster - it overflows when screen is smaller than 600px.
  2. Which layout adjusts automatically?
    Flexible layout (fr) adjusts automatically to any screen size.
  3. Why is fixed layout risky?
    It can overflow, cause horizontal scrolling, and break on mobile devices.
  4. Where can fixed columns be useful?
    Sidebars, navigation menus, or elements that need exact width.
  5. Which unit would you choose for responsive website?
    fr unit - it's flexible and works on all screen sizes.

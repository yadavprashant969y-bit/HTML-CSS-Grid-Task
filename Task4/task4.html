Answers

  1. What does auto do in grid rows?
    auto makes the row height adjust based on content size.
  2. Which row grows based on content?
    The row with auto value grows based on content.
  3. Can rows have different heights?
    Yes, each row can have different height (px, auto, fr, etc).
  4. What happens if content is very large?
    auto row expands to fit content, fixed row may overflow.
  5. Where are grid rows useful in real layouts?
    Header/footer (fixed), main content (auto), page layouts.

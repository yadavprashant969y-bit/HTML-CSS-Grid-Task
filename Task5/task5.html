Answers

  1. Where does gap add space?
    Between grid items (rows and columns), not on edges.
  2. How is gap different from margin?
    gap only adds space between items, margin adds space around each item.
  3. Does gap affect container size?
    No, gap only affects spacing between items inside.
  4. Can we use gap only for rows or columns?
    Yes, use row-gap or column-gap separately.
  5. Why is gap preferred in Grid layouts?
    Simpler, no double spacing issues, cleaner code than margin.

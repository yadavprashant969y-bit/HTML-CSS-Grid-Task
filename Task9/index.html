Answers

1. Why is 12-column grid popular?

Because 12 is divisible by 2, 3, 4, and 6, making it extremely flexible for creating different layout combinations (halves, thirds, quarters).

2. How does spanning help layout?

Spanning allows content to take up specific widths relative to the container, creating structure like sidebars avoiding full-width text everywhere.

3. Where is this system used?

It is the standard for most web frameworks like Bootstrap, Tailwind, and professional design systems.

4. Can we create layouts without this system?

Yes, but it might be harder to maintain consistency across different pages without a standard grid.

5. What happens if column count changes?

If you change from 12 to 10 columns, all your expected spans (like span-6 for half) will break or look different because the math changes.
